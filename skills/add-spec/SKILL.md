# Add Spec

1. Read the user's requirement.
2. Find the project's spec docs and build checklist. Read them to determine which spec doc and checklist section the requirement belongs to.
3. Append or update the relevant section in that spec doc.
4. Add a corresponding unchecked `- [ ]` item to the appropriate checklist section.
5. Do NOT implement the requirement — only update docs and checklist.
